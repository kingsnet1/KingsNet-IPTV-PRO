🛰️ KingsNet IPTV PRO

KingsNet IPTV PRO is a modern, high-performance IPTV Player App for Android and Android TV.
Built with the latest Android 15 (API 35) SDK, it provides a beautiful, responsive interface with advanced streaming, EPG, and casting features — ready for production or customization.

🚀 Features
📺 Core Functionality

Live IPTV Channel Playback — powered by ExoPlayer for smooth, adaptive streaming.

XMLTV EPG Parser — parses and displays “Now / Next” program info with automatic updates.

Favorites Management — persist favorite channels using Room Database.

Channel Thumbnails — fast image loading & caching via Coil.

Background Playback — full MediaSession integration for lock-screen and notification controls.

🧠 Advanced Player Features

Multi-audio & subtitle track selection

DVR & Timeshift-ready playback hooks

Ad-injection integration points (Google IMA ready)

Caching and offline preview support

📡 Casting & Multi-Device Support

Chromecast / Google Cast skeleton included (extend with your App ID)

Seamless handoff between devices

Media controls synchronized via Cast session

📺 Android TV (Leanback UI)

Optimized for remote navigation

Channel grid & playback fragment placeholders

EPG integration for live TV experiences

🏗️ Tech Stack
Layer	Technology
UI	Jetpack Compose & Material3 (ready)
Image Loading	Coil
Database	Room Persistence Library
Player Engine	ExoPlayer (v3+)
Networking	Retrofit / XML Parser
EPG Parsing	XMLTV Parser
Casting	Google Cast SDK
TV UI	Android Leanback Support Library
⚙️ Project Setup
🧩 Requirements

1 Android Studio Ladybug or Hedgehog (latest recommended)
2 Min SDK: 24 (Android 7.0)
3 Target SDK: 35 (Android 15)
4 Internet access for IPTV playlist URLs and EPG XML feeds

**🔧 How to Build**
1. Clone this repository

2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or physical device
