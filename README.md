# KingsNet IPTV PRO - Starter Project

This is a starter Android IPTV Player scaffold targeting **Android 15 (API 35)**.

Features included:
- Package: `com.kingsnet.iptvpro`
- App name: `KingsNet IPTV PRO`
- Room DB (channels + favorites)
- Coil for thumbnail loading and caching
- ExoPlayer player with MediaSession hooks
- XMLTV EPG parser with Now/Next helper
- Chromecast integration skeleton (add Cast SDK and App ID)
- Android TV (Leanback) skeleton
- Temporary app icon placeholder included

Open this folder with Android Studio and run a Gradle sync. Add Google Cast SDK dependency and any production configuration (DRM, ads, cast app id) as needed.
